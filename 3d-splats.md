# 3D Splats

[HF:dx8152/Qwen-Image-Edit-2511-Gaussian-Splash](https://huggingface.co/dx8152/Qwen-Image-Edit-2511-Gaussian-Splash)
presents a workflow around [GH:apple/ml-sharp](https://github.com/apple/ml-sharp) tool to create a gaussian splat from a single image
and Qwen 2501 or 2511 edit image generation models. A limitation called out in the associated Youtube video is that
camera can be rotated up to 45 degrees from original point but not more.
